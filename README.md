# 🪔 Diwali Sales Data Analysis using Python

A complete Exploratory Data Analysis (EDA) project on Diwali Sales Dataset using Python. The objective of this project is to uncover customer purchasing behavior, identify high-value customer segments, and derive business insights that can help improve marketing and sales strategies.

---

## 📌 Project Overview

This project analyzes Diwali sales data using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

The analysis focuses on understanding:

- Customer demographics
- Purchasing patterns
- State-wise sales
- Occupation-wise spending
- Product category performance
- Customer segmentation

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

The dataset contains customer purchase information during the Diwali festival, including:

# 📈 Key Business Insights

### 👩 Female customers contribute more revenue than male customers.

---

### 🎯 Customers aged **26–35 years** spend the highest amount during Diwali.

---

### 💍 Married women are the most valuable customer segment based on overall spending.

---

### 🏙️ Top revenue-generating states include:

- Uttar Pradesh
- Maharashtra
- Karnataka

These states contribute significantly to total Diwali sales.

---

### 💼 Customers working in the following professions spend the most:

- IT Sector
- Healthcare
- Aviation

---

### 🛍️ Most purchased product categories include:

- Food
- Clothing & Apparel
- Electronics

These categories receive the highest customer demand during festive sales.

---

### ⭐ Certain products generate significantly higher order volumes than others, indicating strong customer preference.

---

# 📉 Business Recommendations

- Focus festive marketing campaigns on women customers.
- Target the 26–35 age group with personalized offers.
- Prioritize marketing in high-performing states.
- Launch occupation-specific promotions for IT and Healthcare professionals.
- Increase inventory for top-selling product categories before festive seasons.
- Bundle frequently purchased products to increase Average Order Value (AOV).
